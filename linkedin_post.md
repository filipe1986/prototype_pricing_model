I just finished building a Natural Gas Storage Contract Valuation Tool for J.P. Morgan!

Over the last few days I developed a complete desktop application that helps value gas storage contracts by combining:

- Historical price data + linear trend forecasting
- Monthly seasonality modeling
- Real-time contract valuation (buy low in summer → sell high in winter)
- Storage costs, injection/withdrawal rates, and operational constraints
- Clear profitability indicators (green/red) with break-even analysis

### Tech Stack:
- Python 3
- Pandas & SciPy (for forecasting)
- Tkinter (GUI)

The tool is fully functional with an intuitive interface and already handles realistic scenarios — including warning users when a contract is unprofitable due to seasonality.

This project helped me deepen my skills in time series analysis, financial modeling, and building production-ready desktop tools.
